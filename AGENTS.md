# animaciones html

Colección de 15 animaciones HTML con canvas (partículas, galaxias, olas, etc.) diseñadas para fondos de streaming vía OBS (Browser Source).

## Estructura
- `*.html` — Cada archivo es una animación independiente auto-contenida (sin dependencias externas)
- `Cariño - Animaciones Vol. 1.rar` — Respaldo comprimido

## Convenciones
- Canvas 2D fullscreen, fondo negro (`#000`)
- `requestAnimationFrame` para el loop
- `resize` event listener para responsive
- Sin librerías externas (vanilla JS)
- Nombres: `{NN}. {Nombre}.html`

## Animaciones
01. La Nieve — Copos de nieve con parallax lento
02. Galaxia — Partículas en espiral (galaxia rotando)
03. Dunas — Olas de arena sinuosas
04. Locura Estelar — Estrellas erráticas con estelas
05. Vía Láctea — Brazo galáctico con densidad variable
06. La Pesca — Líneas tipo caña de pescar
07. Hipnótico — Patrón psicodélico rotatorio
08. Tríniti — Triángulos en expansión
09. Las Olas — Olas senoidales superpuestas
10. Olas Grandes — Olas amplias y lentas
11. Malla — Grid deformado por ondas
12. Sismo — Líneas sísmicas horizontales
13. Caminos — Líneas que serpentean aleatoriamente
14. Vetas — Vetas orgánicas tipo mármol
15. Viaje Estrellado — Tunnel de estrellas (efecto warp/hipersalto)
16. Malla 3D — Puntos aleatorios animados conectados por líneas en perspectiva 3D
17. Malla 3D Ligera — Versión más sutil con menos puntos y líneas más espaciadas
